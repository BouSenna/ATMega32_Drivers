# AVL-ATMega32 Drivers
These are some drivers for ATMega32 AVR Microcontroller peripherals and commonly used devices.

### Hardware Abstraction Layer Drivers
- [x] LED
- [x] 7-Segments
- [x] LCD
- [ ] Keypad

### Microcontroller Abstraction Layer Drivers
- [x] DIO
- [ ] SPI
- [ ] I2C
- [ ] Timers
- [ ] ADC
- [ ] UART
